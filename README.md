## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally

## Customization

### Update Personal Information

Edit the components to add your personal details:
- **Hero section**: Update your name and tagline in `src/components/Hero.jsx`
- **About section**: Add your bio in `src/components/About.jsx`
- **Skills**: Update your skills in `src/components/Skills.jsx`
- **Projects**: Add your projects in `src/components/Projects.jsx`
- **Contact**: Update contact info in `src/components/Contact.jsx`

### Styling

All styles are in CSS files located in `src/components/`. You can customize:
- Colors in the CSS variables
- Fonts and typography
- Layout and spacing
- Animations and transitions
